gin 是什么

gin是目前Github上Star最多的 GO WEB 框架。

gin web 框架的优点：

  1. star数最多
  2. 简单原则，以最简洁的方式做好某件事,gin 主要把功能聚焦在路由和请求上面
  3. 并发高,在并发压测情况下,吞吐量很高
  4. 分配内存少,在并发压测情况下,分配内存很低。
